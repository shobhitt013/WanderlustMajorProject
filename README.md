## ✨ Features

- 🏠 Browse beautiful property listings
- 🔍 Search listings by city
- 🏷️ Filter listings by categories
- 👤 User Authentication (Sign Up / Login / Logout)
- ➕ Create new listings
- ✏️ Edit your own listings
- 🗑️ Delete your own listings
- ⭐ Add and delete reviews
- 📍 Interactive Mapbox location
- ☁️ Image upload using Cloudinary
- 📱 Fully responsive UI
- 💰 Price display with GST toggle
- ❤️ Modern Airbnb-inspired design

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/shobhitt013/WanderlustMajorProject.git
```

Go into the project

```bash
cd WanderlustMajorProject
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
ATLASDB_URL=YOUR_MONGODB_URI

SECRET=YOUR_SESSION_SECRET

MAP_TOKEN=YOUR_MAPBOX_TOKEN

CLOUD_NAME=YOUR_CLOUDINARY_NAME

CLOUD_API_KEY=YOUR_CLOUDINARY_API_KEY

CLOUD_API_SECRET=YOUR_CLOUDINARY_SECRET
```

Run the project

```bash
node app.js
```

or

```bash
nodemon app.js
```
